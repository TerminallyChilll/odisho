# Odisho Salon

A professional React-based web application for Odisho Salon.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is configured for automatic deployment to GitHub Pages via GitHub Actions. Any push to the `main` branch will trigger a build and update the live site.
